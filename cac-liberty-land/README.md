# CAC Liberty Land Website

## Overview
CAC Liberty Land is a community-focused website designed to provide information about the church, its mission, activities, and ways to connect with the community. The website features a clean and responsive design, ensuring accessibility across various devices.

## Project Structure
The project is organized into the following files and directories:

```
cac-liberty-land
├── index.html        # Main HTML file for the homepage
├── about.html        # Information about the church
├── contact.html      # Contact form and details
├── css
│   └── styles.css    # Styles for the website
├── js
│   └── main.js       # JavaScript for interactive functionality
├── assets
│   └── cac-logo.svg   # Logo for CAC Liberty Land
├── README.md         # Project documentation
└── LICENSE           # Licensing terms
```

## Setup Instructions
1. **Clone the Repository**
   ```
   git clone <repository-url>
   cd cac-liberty-land
   ```

2. **Open the Project**
   Open `index.html` in your web browser to view the homepage.

3. **Modify Styles**
   Edit `css/styles.css` to customize the appearance of the website.

4. **Add Functionality**
   Use `js/main.js` to implement any interactive features or enhancements.

5. **Update Content**
   Modify `about.html` and `contact.html` to provide relevant information about the church and contact details.
